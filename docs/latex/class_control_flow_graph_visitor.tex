\doxysubsection{Detailed Description}
This class constructs the Control Flow Graph (C\+FG) that represents the program\textquotesingle{}s control flow. Note that the condition of the A\+ST objects \char`\"{}\+For\char`\"{}, \char`\"{}\+If\char`\"{}, \char`\"{}\+While\char`\"{} are considered as statements too, although our A\+ST treats them as expressions (\mbox{\hyperlink{class_abstract_expr}{Abstract\+Expr}}). This is required to model the data flow of variables in the condition. The visitor further collects information that is required to build the Data Flow Graph (D\+FG) therefrom.
