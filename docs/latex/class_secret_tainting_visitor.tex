\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_secret_tainting_visitor}{Secret\+Tainting\+Visitor}} traverses through a given A\+ST and marks all Nodes as \char`\"{}tainted\char`\"{} that in some way deal with an encrypted variable. \mbox{\hyperlink{class_for}{For}} example, if \mbox{[}a,b\mbox{]} are encrypted variables and \char`\"{}int result = a + 2;\char`\"{} is a statement, then the \mbox{\hyperlink{class_var_assignm}{Var\+Assignm}} object, the \mbox{\hyperlink{class_arithmetic_expr}{Arithmetic\+Expr}} object and both of the literals would be marked as tainted.
