\doxysubsection{Detailed Description}
A node representing the retrieval of a matrix\textquotesingle{}s dimension.

\mbox{\hyperlink{class_if}{If}} the given matrix is known at compile-\/time, the method returns the specified dimension of the matrix when executing the \mbox{\hyperlink{class_evaluation_visitor}{Evaluation\+Visitor}}. \mbox{\hyperlink{class_for}{For}} example, if the 2x3 matrix is defined as M = \mbox{[}3 2 1; 5 3 3\mbox{]}, then \mbox{\hyperlink{class_get_matrix_size}{Get\+Matrix\+Size(\+M, 0)}} returns 2 and \mbox{\hyperlink{class_get_matrix_size}{Get\+Matrix\+Size(\+M, 1)}} returns 3. Requesting any other (non-\/existing) higher dimension returns 0.

\mbox{\hyperlink{class_if}{If}} the matrix is not known at compile-\/time, e.\+g., a function\textquotesingle{}s input parameter then the downstream system generating the F\+HE code (e.\+g., W\+O\+OL) must handle the method\textquotesingle{}s implementation by replacing the function\textquotesingle{}s call by its return value (i.\+e., the requested matrix dimension).
