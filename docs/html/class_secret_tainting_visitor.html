Detailed Description

The SecretTaintingVisitor traverses through a given AST and marks all Nodes as "tainted" that in some way deal with an encrypted variable. For example, if [a,b] are encrypted variables and "int result = a + 2;" is a statement, then the VarAssignm object, the ArithmeticExpr object and both of the literals would be marked as tainted.
