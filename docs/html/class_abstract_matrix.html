Detailed Description

A helper class that allows to access and use all Matrix<T> instances using a unified interface, without needing exact knowledge about the specific template type T.

virtual AbstractMatrix* AbstractMatrix::applyBinaryOperator ( AbstractMatrix *  rhsOperand,
Operator *  os 
)
pure virtual

A helper method that is defined in AbstractMatrix and overridden here. It casts the first parameter (AbstractMatrix) to Matrix<T> where T is the template type (e.g., int, float). This is needed to call the suitable applyBinaryOperatorComponentwise method that handles the binary operator application.

Parameters
rhsOperandThe operand on the right hand-side. The current matrix is the left hand-side operand.
opThe operator to be applied on the two given matrices.
Returns
The AbstractMatrix resulting from applying the operator op on the two matrices.
